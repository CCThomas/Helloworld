## How to use Git
First Add Changes
```
git add <file_to_add>
```

Commit Changes
```
git commit -m "Commit Message"
```
Pull from Branch, helps avoid merge conflicts
```
git pull <branch-name>
```
Finally Push to Branch
```
git push -u origin <branch-name>
```
